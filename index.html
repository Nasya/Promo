Гірський велосипед

У загальному випадку маунтінбайк призначений для їзди по дорогах невисокої якості або поза ними, по місцевості з сильно пересіченим рельєфом. Зручний і при їзді по міських бордюрах. Поняття «гірський велосипед» дуже широке і об'єднує кілька підтипів, які істотно відрізняються за призначенням і конструкцією.

Міські велосипеди

Міські велосипеди призначені для частих, щодо нешвидких поїздок по рівній міській місцевості. На них зазвичай їздять в повсякденному одязі, при будь плюсовій температурі.

Гібрид (HYBRID)

Знадобилося створити якісь проміжні варіанти, які дозволили б поєднати переваги різних типів велосипедів. Традиційно під гібридом мається на увазі полегшений варіант гірського велосипеда краще пристосований для їзди по шосе.

ТУРИНГ (TOURING)

Якщо гібрид - це крок до більш універсального велосипеда від гірського, то турінг є кроком до універсального велосипеда від шосейного. Цей велосипед призначений для тривалих подорожей у швидкому темпі, але з можливістю їзди як по хороших шосе, так і, при необхідності, по дорогах середньої якості.

BMX

Цей тип велосипедів виник на початку 1970-х. Він пристосований для виконання трюків і гонок на спеціальних майданчиках з штучними перешкодами. Але взагалі, BMX підходить не тільки для спорту, але і для звичайного катання.

Шосейний велосипед

Шоссейний велосипед призначений для швидкої їзди по хорошим шосе. На гладкому асфальті на шосейному велосипеді можна тримати швидкість на 7-10 км/год вище, ніж на міському або гірському велосипеді при однакових енерговитратах. Правда, на грубозернистому асфальті перевага буде істотно менше, а на поганій дорозі їхати буде взагалі важко.
